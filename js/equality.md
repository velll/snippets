---
title: Equality operators
---
a equal to b
```js
a == b
```


a equal to b AND type of a is equal to type of b
```js
a === b
```

Examples
```js
0 == false
// -> true

0 === false
// -> false
```

# Negation

a is not equal to b
```js
a != b
```

a is not equal to b OR type of a is not equal to type of b
```js
a !== b
```

Examples
```js
0 != false
// -> false (they are actually equal)

0 !== false
// -> true (they don't have the same type)
```
